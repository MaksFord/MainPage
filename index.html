Commentary

This page was created as the basis for my future portfolio. I wanted to add language switcher and nessessary links list but the main goal was to make the page visually attractive while having multiple color themes as an option. Originally i planned to use different tool for each elements in header, button for language, select tool for the themes and display switcher for the links but ended up with the last for all for aestetic purposes. Select require too many crutches to style properly and i end up using the same option for all because the result simply looked nicer. After main functions were ready I made some optimization for smaller screens.
